Overview

Technical module for providing some utilities and libraries that can be used in other applications can be used in other applications. This module has no direct influence on the running system, but can be used as a dependency in other modules in order to utilise the corresponding functions.

Icon Selection Field

A selection field which offers the option of assigning a corresponding icon to the various options. If the field is used in the list view with the noLabel attribute, only one icon is displayed, otherwise the icon is displayed to the left of the selected text 

<field 
	name="company_type" 
	widget="selection_icons" 
	options="{'icons': {'person': 'user', 'company': 'building'}}"
/>

Icon Text Field

A text widget which can be used for char, text and html fields. Instead of displaying the content of the field, an icon is displayed which shows a popover with the content when clicked. This widget has only a readonly mode and is ideal for displaying text fields as information in the list view. 

<field 
	name="comment" 
	widget="text_icon" 
/>
